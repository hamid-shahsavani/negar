## *description*
  + #### *text*
    ```python
      # write 'text' to log file
      text = 'negar' 
    ```
  + #### *save*
    ```python
      # write log to 'negar-log.txt'
      save = 'negar-log.txt' 
    ```
  + #### *size*
    ```python
      # set 'size' for log file 
      size = 2
    ```
